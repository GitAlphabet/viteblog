### whistle

#### 1、安装/启动

```bash
# 安装
sudo npm install -g whistle

# 查看帮助
w2 help

# 启动
w2 start

# 重启whsitle:
w2 restart

# 停止whistle:
w2 stop

# 调试模式启动whistle(主要用于查看whistle的异常及插件开发)
w2 run
```

#### 2、访问 Whistle

```bash
address：127.0.0.1:8899
```

#### 3、配置代理

```bash
手机(需要和电脑在同一网段)配置代理即可抓包。

```
